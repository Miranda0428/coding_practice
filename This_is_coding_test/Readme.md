# Coding_practice
Coding Test Practice

문제를 풀때는 일반화하여 Rule을 먼저 발견한다.
Rule을 세우고 나면 이것을 구현하는 것은 어렵지 않다.

문제를 풀면서 기억해야 하는 사항을 반드시 주석으로 남긴다.
주석은 나중에 오답노트처럼 쓰인다.
